# My Dotfiles

**Goals**
* Systemwide font configuration
* Systemwide color configuration
* Syntaxbased color consistency through base16

**Components**
* qtile
* qutebrowser
* kitty
* vim

**Fonts**
* fonts-mononoki
* fonts-font-awesome

**Quick Setup**

    sudo apt install git
    git clone --recurse-submodules https://github.com/ura43/dotfiles
    cd dotfiles
    sudo ./install
