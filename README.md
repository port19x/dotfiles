# My Dotfiles

**Purposes**
* Portability of configuration
* Sharing is caring

**Goals**
* Systemwide font configuration
* Systemwide color configuration
* Syntaxbased color consistency through base16

**Non Goals**
* No Transparency
* No Animations
* No Distractions

**Components**
* bash
* kitty
* qutebrowser
    * youtube-dl
    * mpv
* vim

* stow (optional)

**Fonts**
* fonts-mononoki
* fonts-font-awesome

**Quick Setup**

    sudo apt install git

    git clone --recurse-submodules https://github.com/ura43/dotfiles

    cd dotfiles

    sudo ./install
